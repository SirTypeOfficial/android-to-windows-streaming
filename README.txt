========================================
Android to Windows Streaming
استریم اندروید به ویندوز
========================================

استفاده از دوربین و میکروفون گوشی اندروید به عنوان WebCam در ویندوز

========================================
نصب سریع
========================================

1. Windows App:
   - اجرای setup.bat در پوشه windows-app
   - اجرای run.bat برای شروع

2. Virtual Camera:
   - اجرای build.bat در پوشه virtual-camera (به عنوان Admin)
   - اجرای register.bat (به عنوان Admin)

3. Android App:
   - باز کردن پروژه در Android Studio
   - Build و اجرا روی دستگاه

========================================
استفاده سریع
========================================

1. اپلیکیشن ویندوز را باز کنید
2. اپلیکیشن اندروید را باز کنید
3. "شروع استریم" را در اندروید بزنید
4. آدرس IP را در ویندوز وارد کنید
5. "اتصال WiFi" یا "اتصال USB" را بزنید
6. "فعال‌سازی دوربین مجازی" را بزنید
7. از "Android Virtual Camera" در برنامه‌های دیگر استفاده کنید

========================================
فایل‌های مهم
========================================

- OVERVIEW.txt: خلاصه کامل پروژه
- SETUP_GUIDE.txt: راهنمای جامع نصب
- PROJECT_STRUCTURE.txt: ساختار دقیق پروژه
- TROUBLESHOOTING.txt: راهنمای عیب‌یابی
- CHANGELOG.txt: تاریخچه تغییرات

========================================
ویژگی‌ها
========================================

✓ استریم ویدیو HD (تا 1080p)
✓ استریم صدا استریو
✓ اتصال WiFi و USB
✓ کنترل کامل دوربین
✓ Virtual Camera برای Zoom/Teams/Skype
✓ تاخیر بسیار کم
✓ رابط کاربری ساده

========================================
پیش‌نیازها
========================================

Android:
- Android 8.0+ (API 26+)
- 100 MB فضای خالی

Windows:
- Windows 10/11
- Python 3.11+
- Visual Studio 2022 (برای Virtual Camera)
- 500 MB فضای خالی
