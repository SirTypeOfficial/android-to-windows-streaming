Android to Windows Streaming System
تاریخچه تغییرات
========================================

نسخه 1.0 - اکتبر 2025
========================================

اولین نسخه عمومی

ویژگی‌های اضافه شده:

Android App:
- پیاده‌سازی CameraX برای دسترسی به دوربین
- انکودر H.264 با استفاده از MediaCodec
- انکودر AAC برای صدا
- سرور Socket برای اتصال WiFi
- پشتیبانی از اتصال USB با ADB
- رابط کاربری Material Design
- سرویس Foreground برای استریم در پس‌زمینه
- کنترل دوربین (سوییچ، فلش، فوکوس)
- پشتیبانی از رزولوشن‌های متعدد
- نمایش وضعیت اتصال
- کد اتصال 6 رقمی

Windows App:
- رابط کاربری PyQt6
- مدیریت اتصال WiFi و USB
- دیکودر ویدیو و صدا با PyAV
- نمایش لحظه‌ای ویدیو
- پخش صدا
- ارسال دستورات کنترلی
- رابط با Virtual Camera
- نمایش FPS و Latency
- تنظیمات رزولوشن و FPS

Virtual Camera Driver:
- DirectShow Source Filter
- پشتیبانی از RGB24
- Shared Memory IPC
- قابلیت ثبت/حذف خودکار
- سازگار با Windows 10/11
- کار با تمام برنامه‌های استاندارد

Protocol:
- پروتکل binary با header ثابت
- پشتیبانی از انواع پکت مختلف
- دستورات کنترلی JSON
- Timestamp برای همگام‌سازی
- Sequence number برای ترتیب پکت‌ها

Documentation:
- راهنمای کامل نصب
- مستندات ساختار پروژه
- راهنمای استفاده
- راهنمای عیب‌یابی

Scripts:
- اسکریپت نصب خودکار Windows
- اسکریپت Build Virtual Camera
- منوی Quick Start
