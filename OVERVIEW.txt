========================================
Android to Windows Streaming System
نظام استریم اندروید به ویندوز
========================================

نسخه: 1.0
تاریخ: 2025

========================================
خلاصه پروژه
========================================

این پروژه یک سیستم کامل استریم ویدیو و صدا از دستگاه اندروید به کامپیوتر ویندوز است
که امکان استفاده از دوربین و میکروفون گوشی به عنوان یک WebCam مجازی در ویندوز را فراهم می‌کند.

ویژگی‌های کلیدی:
- استریم ویدیو با کیفیت HD (تا 1080p)
- استریم صدا استریو
- اتصال از طریق WiFi یا USB
- کنترل کامل دوربین از ویندوز
- Virtual Camera برای استفاده در Zoom، Skype، Teams و...
- تاخیر بسیار کم (Low Latency)

========================================
معماری سیستم
========================================

      [Android Device]

========================================
تکنولوژی‌های استفاده شده
========================================

Android:
- Kotlin
- CameraX (Camera API)
- MediaCodec (Hardware Encoding)
- Socket Programming
- Android Services

Windows:
- Python 3.11+
- PyQt6 (UI)
- PyAV (FFmpeg wrapper)
- OpenCV (Image Processing)
- PyAudio (Audio Playback)

Virtual Camera:
- C++17
- DirectShow API
- Windows Media Foundation
- COM (Component Object Model)
- Shared Memory IPC

========================================
فرمت‌های رمزگذاری
========================================

Video:
- Codec: H.264 (AVC)
- Profile: Baseline
- Bitrate: 2 Mbps (قابل تنظیم)
- Resolution: 480p/720p/1080p
- FPS: 15/30/60

Audio:
- Codec: AAC-LC
- Sample Rate: 44.1 kHz
- Channels: Stereo (2)
- Bitrate: 128 kbps

========================================
روش‌های اتصال
========================================

1. WiFi:
   - سرور در اندروید راه‌اندازی می‌شود
   - ویندوز به IP گوشی متصل می‌شود
   - پورت پیش‌فرض: 8888
   - مناسب برای شبکه‌های محلی

2. USB:
   - از ADB reverse استفاده می‌کند
   - اتصال مستقیم از طریق کابل USB
   - پورت پیش‌فرض: 8889
   - تاخیر کمتر و پایدارتر

========================================
امکانات کنترلی
========================================

از ویندوز می‌توانید موارد زیر را کنترل کنید:
- تغییر رزولوشن (480p, 720p, 1080p)
- تغییر نرخ فریم (15, 30, 60 FPS)
- سوییچ بین دوربین جلو و عقب
- روشن/خاموش کردن فلش
- تنظیم روشنایی
- تنظیم فوکوس

========================================
Virtual Camera Driver
========================================

این درایور یک DirectShow Source Filter است که:
- به عنوان دوربین مجازی در ویندوز ثبت می‌شود
- از Shared Memory برای دریافت فریم استفاده می‌کند
- فرمت RGB24 را پشتیبانی می‌کند
- در تمام برنامه‌های استاندارد ویندوز کار می‌کند

========================================
نصب سریع
========================================

1. Windows App:
   cd windows-app
   setup.bat
   run.bat

2. Virtual Camera:
   cd virtual-camera
   build.bat (as Admin)
   register.bat (as Admin)

3. Android App:
   - باز کردن پروژه در Android Studio
   - Run on device/emulator

یا استفاده از quick_start.bat در root پروژه

========================================
استفاده سریع
========================================

1. اپلیکیشن ویندوز را اجرا کنید
2. اپلیکیشن اندروید را اجرا کنید
3. در اندروید روی "شروع استریم" کلیک کنید
4. در ویندوز آدرس IP گوشی را وارد کنید
5. روی "اتصال WiFi" یا "اتصال USB" کلیک کنید
6. پس از اتصال، "فعال‌سازی دوربین مجازی" را بزنید
7. در برنامه مورد نظر، "Android Virtual Camera" را انتخاب کنید

========================================
پیش‌نیازها
========================================

Android:
- دستگاه اندروید با API 26+ (Android 8.0+)
- Android Studio (برای Build)
- 100 MB فضای خالی

Windows:
- Windows 10/11 (64-bit)
- Python 3.11 یا بالاتر
- 500 MB فضای خالی
- Visual Studio 2022 (برای Build Virtual Camera)
- ADB (برای اتصال USB - اختیاری)

========================================
محدودیت‌ها و نکات
========================================

محدودیت‌ها:
- رمزگذاری اتصال وجود ندارد (برای محیط محلی امن است)
- فقط یک client می‌تواند به اندروید متصل شود
- Virtual Camera در Linux/Mac کار نمی‌کند
- حداکثر رزولوشن 1080p

نکات عملکردی:
- برای تاخیر کمتر از USB استفاده کنید
- برای صرفه‌جویی در باتری، رزولوشن و FPS را کاهش دهید
- در صورت مشکل، دستگاه اندروید را restart کنید
- فایروال ویندوز ممکن است باید تنظیم شود

========================================
